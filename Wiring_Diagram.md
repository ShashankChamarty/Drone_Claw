            +---------------------- 7.4V BATTERY -----------------------+
            |                                                           |
            |   (+) ----------------------------> Breadboard + Rail     |
            |   (-) ----------------------------> Breadboard - Rail     |
            +-----------------------------------------------------------+


                 +============= BREADBOARD =============+
                 |                                      |
                 |   + Rail  ========================   |
                 |   - Rail  ========================   |
                 |                                      |
                 |   Capacitor (+) -> + Rail            |
                 |   Capacitor (-) -> - Rail            |
                 |                                      |
                 |   + Rail  -> Arduino VIN             |
                 |   - Rail  -> Arduino GND             |
                 |                                      |
                 |   + Rail  -> Servo VCC (Red)         |
                 |   - Rail  -> Servo GND (Black/Brown) |
                 +======================================+


            +----------------- Arduino Nano ESP32 -------------------+
            |                                                        |
            |   D5  ---------------------------------> Servo Signal  |
            |                                                        |
            |   VIN <--------------------------------- + Rail        |
            |   GND <--------------------------------- - Rail        |
            +--------------------------------------------------------+

            +--------------- SERVO: LDX-335MG ----------------+
            |              (CLAW MECHANISM ACTUATOR)          |
            |                                                 |
            |  White  <--------------- D5 (Signal)            |
            |  Red    <--------------- + Rail (VCC)           |
            |  Black  <--------------- - Rail (GND)           |
            |                                                 |
            |  Function: Opens and closes claw mechanism      |
            +-------------------------------------------------+
